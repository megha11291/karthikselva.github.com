Published on: Feb 23rd, 2010
Tags: Linux, Projects

tested with :

i) ubuntu 8.10

ii)gnokii

iii) mysql 5

iv) lg kp 199

functionalities :

i) just message ”lock” to your server mobile to lock the screen

ii)   ”unlock” to release the screen
